HyperText Markup Language (HTML) is used to create web pages using elements such as headings, paragraphs, and links. With HTML, you learn about the different types of HTML elements and how to use them.
Cascading Style Sheets (CSS) are used to change the appearance of web pages, such as colors, fonts, and layouts. With CSS, you can enhance and organize your web page.
JavaScript is used to make web pages interactive, such as handling click events and understanding basic programming concepts like variables and functions.
